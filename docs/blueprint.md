# **App Name**: FarmFlow

## Core Features:

- Farmer Registration & Login: Secure farmer account creation and login functionality using a database for storing user credentials and profiles.
- Farm Listing Dashboard: Display a personalized dashboard listing all farms associated with the logged-in farmer, retrieved from the database.
- Add New Farm: Provide a form for farmers to add new farm records, including details such as farm name, location, and key characteristics, which are saved to the database.
- Farm Detail & Crop History: View comprehensive details for a selected farm, including previously cultivated crops and relevant historical data stored in the database.
- AI-Powered Crop Recommendation Tool: A button on the farm detail page that, when activated, initiates an AI tool to suggest optimal crop types for that specific farm based on user-defined inputs or existing farm data (e.g., location, climate, soil conditions).

## Style Guidelines:

- Primary color: A muted yet vibrant green (#248B3D) evoking nature, growth, and productivity. This provides strong contrast for readability on light backgrounds.
- Background color: A very light, almost off-white green (#F0F6F1) for a clean, fresh, and expansive feel.
- Accent color: A brighter, contrasting yellow-green (#85E02D) to draw attention to interactive elements, calls to action, and important highlights, symbolizing freshness and vitality.
- Headline and body font: 'Inter', a neutral and highly readable sans-serif typeface, providing a modern, objective, and clear aesthetic for both data presentation and user interactions.
- Use clear, recognizable, and minimalist line-art icons that directly relate to farming activities, crops, soil, and management tasks, maintaining a cohesive visual language.
- Adopt a clean and functional dashboard-style layout, prioritizing clear navigation and intuitive organization of farm information. Implement responsive design principles to ensure optimal viewing and interaction across various devices.
- Incorporate subtle, functional animations for feedback on user actions (e.g., successful submission, data loading indicators) and smooth transitions between pages or states, enhancing the user experience without distraction.
